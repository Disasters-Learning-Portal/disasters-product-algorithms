# =============================================================================
# Canonical DEV-LOCAL conda dep list
# =============================================================================
# Single source of truth for the conda packages needed to develop / lint /
# test this repo OUTSIDE the hub image (e.g. on a laptop or in CI).
#
# Used by:
#   - .github/workflows/lint.yml (the cli-smoke job reads this file)
#   - The "How to install for dev" instructions in pyproject.toml and README
#
# Format: one conda package spec per non-comment line, exactly as it would
# appear to `mamba install -c conda-forge` (e.g. "gdal" or "rasterio>=1.3").
# Lines starting with # are ignored. Blank lines are ignored.
#
# Adding a new dep:
#   - Pip-installable (manylinux wheel) → prefer pyproject.toml [project.dependencies].
#   - Conda-only / native lib / GDAL plugin → add a line here.
#   - If the hub image also needs the new conda dep ON TOP of Pangeo base,
#     ALSO add it to hub-conda-deps.txt (which is auto-synced to the image
#     repo by .github/workflows/sync-conda-deps.yml). hub-conda-deps.txt
#     covers deps that aren't already in Pangeo base — most things below
#     ARE in Pangeo base, hence the difference between the two files.
# =============================================================================
